# 🌍 ระบบตรวจสอบสภาพแวดล้อม IoT Environment Monitoring System

## 📌 รายละเอียด
เป็นระบบตรวจสอบสภาพแวดล้อมแบบเรียลไทม์ด้วยเทคโนโลยี IoT โดยสามารถตรวจวัดค่าอุณหภูมิ ความชื้น ฝุ่น และกลิ่น พร้อมแสดงสถานะคุณภาพอากาศผ่านหน้าเว็บ (Web Dashboard) และแจ้งเตือนผู้ใช้งานผ่าน LINE Official Account เมื่อค่าต่าง ๆ เกินเกณฑ์ที่กำหนด

---

## 🎯 วัตถุประสงค์
- เพื่อพัฒนาระบบ IoT สำหรับตรวจวัดสภาพแวดล้อมแบบเรียลไทม์ เช่น อุณหภูมิ ความชื้น และคุณภาพอากาศ  
- เพื่อให้ผู้ใช้งานสามารถตรวจสอบข้อมูลได้จากทุกที่ผ่านระบบออนไลน์  
- เพื่อเพิ่มความสะดวกในการติดตามและควบคุมสภาพแวดล้อมโดยไม่ต้องตรวจสอบหน้างาน  
- เพื่อสร้างระบบแจ้งเตือนอัตโนมัติเมื่อค่าต่าง ๆ เกินเกณฑ์ที่กำหนด  

---

## 🚀 ฟีเจอร์หลัก
- 🌡️ ตรวจวัดข้อมูลแบบเรียลไทม์ (อุณหภูมิ ความชื้น ฝุ่น กลิ่น)  
- 📊 หน้า Dashboard แสดงภาพรวมของข้อมูลในแต่ละอาคาร  
- 🏢 ระบบแสดงข้อมูลแยกตามห้อง  
- 🔍 ค้นหาและตรวจสอบสถานะของแต่ละห้อง  
- 📈 แสดงกราฟข้อมูลย้อนหลังของแต่ละห้อง  
- ⚙️ ระบบ Admin สำหรับจัดการห้องและกำหนดค่า Threshold  
- 🟢 แสดงสถานะคุณภาพอากาศ (เขียว = ปกติ / 🔴 แดง = เกินค่าที่กำหนด)  
- 📲 แจ้งเตือนผ่าน LINE OA เมื่อค่าผิดปกติ  

---

## 🏗️ โครงสร้างการทำงานของระบบ

Sensor / Node-RED  
→ `update_sensor.php` (บันทึกข้อมูล)  
→ ฐานข้อมูล MySQL  
→ `api.php` (ประมวลผลข้อมูล)  
→ Web Dashboard (HTML/CSS/JS)  
→ LINE OA (แจ้งเตือนผ่าน webhook)

## 🛠️ เทคโนโลยีที่ใช้
- Raspberry Pi (ประมวลผลและจำลองข้อมูล)  
- Node-RED (จัดการการไหลของข้อมูล)  
- MySQL (ฐานข้อมูล ผ่าน phpMyAdmin)  
- PHP (พัฒนา Backend API)  
- HTML, CSS, JavaScript (พัฒนา Frontend)  
- LINE Official Account (ระบบแจ้งเตือน)  

---

## 👨‍💻 วิธีการใช้งาน

### 👩‍💼 Admin
- เข้าสู่ระบบผ่านหน้าเว็บไซต์  
- เพิ่ม / ลบ / แก้ไข ห้องในหน้า Setting  
- กำหนดค่า Limit ของแต่ละห้อง  
- ตรวจสอบข้อมูลแบบเรียลไทม์และย้อนหลังผ่าน Dashboard  

### 👤 ผู้ใช้งาน (นักศึกษา, อาจารย์)
- รับการแจ้งเตือนผ่าน LINE OA เมื่อค่าผิดปกติ  
- ตรวจสอบสถานะสภาพแวดล้อมของห้อง  

---

## 🧪 ตัวอย่างการใช้งาน

**Input:** เพิ่มห้อง "5-901"  
**Output:**  
- ห้องถูกเพิ่มในระบบ  
- แสดงข้อมูลสภาพแวดล้อมแบบเรียลไทม์  
- ปรากฏใน Dashboard และสามารถตรวจสอบย้อนหลังได้  

---

## 📊 ผลลัพธ์ของโครงการ
ระบบสามารถตรวจสอบสภาพแวดล้อมได้แบบเรียลไทม์ และสามารถแจ้งเตือนผู้ใช้งานได้อัตโนมัติ ช่วยเพิ่มประสิทธิภาพในการดูแลพื้นที่ เช่น ห้องเรียน อาคาร หรือสำนักงาน
